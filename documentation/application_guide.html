Kite Laundry Builder Application Guide 🛠️

Application Overview

The Kite Laundry Builder is a Flask-based web application that generates custom kite laundry designs, patterns, and construction instructions.

Purpose

  • Design Creation: Interactive forms for custom laundry designs
  • Pattern Generation: Mathematical pattern calculations
  • Instruction Generation: Step-by-step construction guides
  • Material Calculation: Fabric and resource requirements

Data Flow

1. User Input Phase

User → Web Form → Parameter Validation → Geometry Selection

2. Processing Phase

Valid Parameters → Pattern Generator → Mathematical Calculations → Template Creation

3. Output Phase

Generated Pattern → Instruction Assembly → Web Display → Download Options

Core Components

Flask Application (app.py)

  • Routes: Handles web requests and form submissions
  • Templates: Renders HTML pages with dynamic content
  • Static Files: CSS, JavaScript, and images

Pattern Generators (Core/pattern_generators/)

  • Tube Generator: Cylindrical patterns
  • Cone Generator: Conical drogue patterns
  • Bol Generator: Rotor cylinder patterns
  • Flag Generator: Rectangular flags
  • Banner Generator: Long horizontal banners

Usage Workflow

1. Start Application

./run.sh

Access at: http://127.0.0.1:5001

2. Design Creation

  1. Select geometry type
  2. Enter dimensions and parameters
  3. Choose colors and materials
  4. Submit form for processing
